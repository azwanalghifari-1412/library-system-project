# Library Management System  
Backend REST API menggunakan Node.js, Express.js, dan SQLite

## Deskripsi
Library Management System ini dibangun sebagai backend dasar untuk mengelola koleksi buku di perpustakaan.  
Tujuan utama project ini adalah mempelajari implementasi REST API, routing, controller, struktur folder yang bersih, serta integrasi database SQLite dalam aplikasi Node.js.

## Fitur Utama
- Tambah buku baru
- Tampilkan seluruh buku
- Pinjam buku (borrow)
- Update status peminjaman
- Penyimpanan data menggunakan SQLite

## Teknologi
- Node.js
- Express.js
- SQLite (sqlite3)

## Status Project
Project ini dikembangkan secara bertahap (incremental commits).
